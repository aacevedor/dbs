Proyectos vs. aplicaciones

¿Cuál es la diferencia entre un proyecto y una aplicación? Una aplicación es una aplicación web que hace algo
por ejemplo, un sistema de Weblog, una base de datos de los registros públicos o una aplicación sencilla encuesta.
Un proyecto es un conjunto de configuración y aplicaciones para un sitio web en particular. Un proyecto puede contener
varias aplicaciones. Una aplicación puede estar en varios proyectos.
